    SGRT File Data File Format

    메타 데이터의 끝으로 건너뛰기
    • 작성자 : 함상식, 최근 변경 : 제갈욱 조금 전
    메타 데이터의 시작으로 이동

    Protocol Data Structure

    Name

    Type

    Description


    Protocol

    Version



    Int

    Protocol 버전 (현재 2)

    Name



    String

    Protocol 이름

    Product



    String

    Protocol 제품명 ( NOTE : Protocol 이 지원하는 product 들, List<String> 으로 변경 필요 )

    Volume



    Int

    Sample의 양 (ul)

    Time



    Int

    전체 PCR 진행 시간 (seconds)

    Date



    String

    생성 시간

    ("yyyyMMdd_HHmmss")

    StepCount



    Int

    전체 Step 개수

    Steps



    List


    Step


    Int

    Step 번호 (1부터 시작)

    Beep


    Bool

    비프음 발생 유무

    AvailGotoStep


    Int

    DefaultStep에서 GotoStep으로 설정 가능한 맨 앞 Step

    DefaultStep에서 GotoBegin, GotoMid가 true 이면 0 (GotoBegin이 true이나 GotoStep이 0이 아닌 경우는 제외)

    MeltStep, GradientStep은 0

    StepType



    Step 종류

    DefaultStep : 0

    MeltStep : 1

    Gradient : 2

    DefaultStep


    Object

    DefaultStep이 아니면 null

    Temperature

    Double

    Step의 PCR 목표 온도 (°C)

    Duration

    Int

    Step 유지 시간 (seconds)

    ReadPlate

    Bool

    Step의 RFU DATA 측정 유무, Camera Capture 유무

    GotoStep

    Int

    Cycle 반복의 시작 Step의 번호

    GotoStepBegin

    Bool

    Cycle의 시작 Step

    GotoStepMid

    Bool

    Cycle 중간 Step

    RepeatTimes

    Int

     반복 Cycle 횟수

    RampRate

    Double

    초당 온도 변화량

    MeltStep


    Object

    MeltStep이 아니면 null

    Start

    Double

    Melt 시작 온도 온도 (°C)

    End

    Double

    Melt 종료 온도 온도 (°C)

    Increment

    Double

    Melt 단계별 온도 증가량

    Duration

    Int

     Melt 온도 변경 후 유지 시간 (seconds)

    ReadPlate

    Bool

    Melt Step의 RFU DATA 측정 유무, Camera Capture 유무

    GradientStep


    Object

    GradientStep이 아니면 null

    Low

    Double

    Gradient 낮은 온도 온도 (°C)

    High

    Double

    Gradient 높은 온도 온도 (°C)

    Duration

    Int

    Step 유지 시간(seconds)













    Plate Data Structure

    Name

    Type

    Description


    Plate

    Version



    Int

    Plate 버전 (2)


    Name



    String

    Plate 이름


    PlateType



    Int

    Plate 종류

    8 strip cap : 1

    96 plate film : 2

    96 plate cap : 3


    WellCount



    Int

    Plate well 개수 (현재 96)


    Date



    String

    생성 시간

    ("yyyyMMdd_HHmmss")


    Dyes



    List<int>

    사용할 Dye 정보

    FAM : 0x11

    HEX : 0x21

    Cal Red 610 : 0x41

    Quasar 670 : 0x81

    Quasar 705 : 0xF1


    Wells



    List




    Position


    String

    Well의 위치

    96 well인 경우

    8x12 행렬로

    열 : 01 부터 12 (D2)

    행 : A 부터 H (upper case)

    A01, A02 ~ H01, H12



    Type


    Int

    Sample 종류

    Sample : 0

    PC = 1

    NC = 2

    Standard(STD) = 3

    NTC = 5

    NRT = 6



    Name


    String

    User 설정한 well 이름



    PatientId


    String

    User가 설정한 Well ID



    ExcluedAnalysis


    Bool

    분석에서 제외할 well



    Dyes


    List





    Channel

    Int

    Channel 번호

    1 ~ 5




    Dye

    Int

    형광 종류

    FAM : 0x11

    HEX : 0x21

    Cal Red 610 : 0x31

    Quasar 670 : 0x41

    Quasar 705 : 0xF1




    Concentration

    double

    Standard Well인 경우의 농도




    Targets

    List<string>

    타겟의 이름으로 낮은 온도 순













    Result Data Structure

    Name

    Type

    Description

    Result

    Version




    Int

    Result 버전 (2)

    Name




    string

    결과 이름

    Tag




    string

    실험 구분을 위한 Unique Id (UUID, GUID 등)

    Protocol




    Object

    표 4.1.1 참조

    Plate




    Object

    표 4.1.2 참조

    Run Info

    Duration



    Int

    전체 PCR 진행된 시간(Seconds)

    ExperimentDate



    String

    PCR 검사 완료 시간("yyyyMMdd_HHmmss")

    ExperimentName



    String

    PCR 실험 이름

    ProductName



    String

    실험에 사용된Protocol의 Product

    ProtocolName



    String

    실험에 사용된Protocol의 Name

    SerialNum



    String

    PCR 장비 Serial 번호

    Status



    Int

    PCR 완료 상태

    비정상 종료 : 0

    정상 완료 : 1

    User Cancelled : 2

    Data




    List


    Channel



    Int

    Channel

    1 ~ 5

    Dye



    int

    Dye 정보

    FAM : 0x11

    HEX : 0x21

    Cal Red 610 : 0x41

    Quasar 670 : 0x81

    Quasar 705 : 0xF1

    StepData



    List



    Step



    RFU data를 측정한 Step


    StepType


    int

    Step 종류

    DefaultStep : 0

    MeltStep : 1

    Gradient : 2


    Temperature


    List<double>

    RFU를 측정한 step의 유지 온도

    DefaultStep인 경우 온도 값을 한 개 갖는 리스트

    MeltStep인 경우에는 RFU를 측정한 온도가 리스트로 있음

    (ex, [60, 60.5, ….])


    WellData


    List




    Well

    string

    Well Position 값으로 96 well의 경우 A01 ~ H12중의 값을 갖는다



    Quantification

    List<double>

    측정한 RFU data 리스트

    DefaultStep은 cycle 순서로 MeltStep은 온도 순으로 있음
